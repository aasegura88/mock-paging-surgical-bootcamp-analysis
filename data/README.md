# Data folder

Expected input file:

```text
data/raw/Mock_Page_data_cleaned.csv
```

This repository is designed so that code can be shared without sharing restricted participant-level survey data.

Recommended practice:

- do not commit the raw CSV unless you have explicit approval
- keep `data/raw/` local and ignored by Git
- if sharing data publicly is not allowed, provide a brief data dictionary or a synthetic example dataset instead
- state in the manuscript and README how qualified readers can request access, if applicable
